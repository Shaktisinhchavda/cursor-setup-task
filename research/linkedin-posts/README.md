# LinkedIn Posts

This folder will hold LinkedIn posts organized by author.

Planned structure:

```text
author-name/
  post-2026-06-16-topic.md
```

Each post file should include:

- Author
- LinkedIn profile URL
- Original post URL
- Date posted, if visible
- Date collected
- Post text or short excerpt
- Research note explaining why it matters

LinkedIn often blocks unauthenticated scraping. Manual collection is acceptable if each post URL and collection date is recorded.
