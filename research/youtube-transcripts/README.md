# YouTube Transcripts

This folder will hold transcripts organized by video.

Planned file format:

```text
expert-name_video-title_or-video-id.md
```

Each transcript file should include:

- Expert or channel
- Video title
- Video URL
- Date collected
- Transcript source
- Transcript text or collection notes

No transcript text should be added unless it was actually collected from a transcript API, public captions, or manual transcript export.
